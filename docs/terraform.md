# Launch EC2 instance

# Login to EC2

# Configure AWS

`aws configure`

# Install Terraform

`wget https://releases.hashicorp.com/terraform/0.12.5/terraform_0.12.5_linux_amd64.zip`

`unzip terraform_0.12.5_linux_amd64.zip`

`mv terraform /usr/bin/.`


# Verify terraform

`terraform version`

# Terrafom commands

`terraform validate`

`terraform init`

`terraform plan`

`terraform apply`

`terraform apply -auto-approve`

`terraform destroy`
